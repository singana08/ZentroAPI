ğŸš¨ COMMON ISSUES & QUICK FIXES

Issue: "Connection to database failed"
  Fix: 1. Verify PostgreSQL is running
       2. Check connection string in appsettings.json
       3. Try: psql -U postgres (to test connection)
  Details: See GETTING_STARTED.md

Issue: "Email not sending"
  Fix: 1. Verify SMTP settings
       2. For Gmail: Use App Password (not regular password)
       3. Enable 2-Step Verification on Google Account
  Details: See QUICK_START.md

Issue: "JWT Token validation failed"
  Fix: 1. Verify secret key hasn't changed
       2. Check token format: "Bearer <token>"
       3. Check Authorization header is present
  Details: See API_ENDPOINTS.md

Issue: "Port 5001 already in use"
  Fix: 1. Change port in launchSettings.json
       2. Or: Stop other services using port 5001

Issue: "Build failed"
  Fix: Run: dotnet clean && dotnet restore && dotnet build
